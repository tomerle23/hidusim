# אפליקציית "חידושי" - בית מדרש דיגיטלי אישי ושיתופי

ברוך הבא לאפליקציית "חידושי" - פלטפורמה מודרנית ויוקרתית לקריאה, כתיבה, העלאה ומחקר של חידושי תורה, סוגיות בגמרא וביאורי הלכה. האפליקציה פותחה במיוחד כדי לשלב את המבנה והערך של עולם הלימוד התורני עם ממשק דיגיטלי מתקדם, רספונסיבי ומעוצב לעילא.

---

## 🛠️ טכנולוגיה ומבנה הפרויקט
האפליקציה נבנתה כ-Single Page Application (SPA) מלאה המבוססת על טכנולוגיות אינטרנט טהורות ללא תלות בשרת:
* HTML5 לשלד המבנה.
* Vanilla CSS לעיצוב יוקרתי (מצבי יום/לילה, זכוכית מטושטשת ואנימציות עדינות).
* Vanilla JS (ES6) לניהול מצב, חישובי גימטריה, פענוח קבצים דינמי ואינטראקטיביות.

### קבצי הפרויקט:
1. index.html - דף המבנה הראשי המכיל את כל מסכי האפליקציה.
2. index.css - מערכת העיצוב (צבעים, פונטים, רספונסיביות).
3. parser.js - מנוע פענוח חכם שיודע לקרוא קובצי טקסט תורניים, לזהות פסוקים, פירושי פרד"ס (פשט, רמז, דרש, סוד) וגימטריה.
4. app.js - מוח האפליקציה המנהל את הנתונים, ה-localStorage, הניווט ומחשבון הגימטריה.
5. פרשת ואתחנן חלק א.txt & פרשת ואתחנן חלק ג.txt - קובצי התוכן שלך שנטענים דינמית ישירות למערכת.

---

## 🚀 איך להפעיל את האפליקציה במחשב המקומי?
מכיוון שהאפליקציה סטטית לחלוטין, ישנן שתי דרכים להפעיל אותה:

### דרך א' (הכי פשוטה):
פשוט לחץ לחיצה כפולה על הקובץ index.html והוא ייפתח ישירות בכל דפדפן (כרום, אדג', ספארי וכו').

הערה: חלק מהדפדפנים המודרניים חוסמים טעינה מקומית של קובצי טקסט חיצוניים (פונקציית fetch מקומית) מטעמי אבטחה (CORS). אם אינך רואה את חידושי פרשת ואתחנן נטענים בדף הבית בדרך זו, השתמש בדרך ב' או ג'.

### דרך ב' (באמצעות שרת מקומי מהיר):
אם מותקן אצלך VS Code, תוכל להפעיל את התוסף Live Server שיפתח את האתר בשרת מקומי בלחיצת כפתור אחת.
או באמצעות הרצת פקודה קצרה בטרמינל (אם מותקן Node/Python):
* ב-Python: python -m http.server 8000 ואז כניסה לכתובת http://localhost:8000.

### דרך ג' (שימוש במעלה הקבצים המובנה):
תוכל לפתוח את index.html ישירות, לעבור למסך "הספרייה שלי", ולגרור את קובצי ה-.txt של פרשת ואתחנן ישירות לתוך אזור העלאת הקבצים! האפליקציה תפענח אותם מיידית בדפדפן ותציג אותם.

## 🌐 איך להעלות את האפליקציה לאינטרנט כאתר עצמאי בחינם?

מכיוון שהאתר הוא סטטי לחלוטין (אין שרת או בסיס נתונים מורכב), העלאתו לרשת היא פשוטה וחינמית לחלוטין:

### אפשרות 1: באמצעות Netlify Drop (הכי פשוטה ומהירה - ללא קוד!)
1. הכנס לאתר Netlify Drop (https://app.netlify.com/drop).
2. גרור את תיקיית הפרויקט המלאה (torah-insights) מהמחשב שלך ושחרר אותה בתוך הריבוע שבמסך.
3. האתר יעלה לאוויר בתוך מספר שניות ותקבל קישור רשמי (לדוגמה: https://torah-insights.netlify.app) שתוכל לשתף עם כולם!

### אפשרות 2: באמצעות GitHub Pages
1. פתח חשבון ב-GitHub (https://github.com) וצור מאגר קוד (Repository) חדש.
2. העלה את קבצי הפרויקט לתוך המאגר.
3. הכנס לכתובת: Settings -> Pages במאגר שלך.
4. תחת "Build and deployment", בחר בענף main ולחץ על Save.
5. האתר שלך יתפרסם בתוך כדקה בקישור ייעודי חינמי.

## ✨ תכונות עיקריות באפליקציה:
* היכל בית המדרש (פיד): סינון לפי קטגוריות (תורה, גמרא, הלכה וכו'), מיון לפי פופולריות או אורך, וחיפוש דינמי.
* עורך הספרים (השולחן שלי): כתיבת חידושים עם תצוגה מקדימה המעוצבת כמו דף ספר קלאסי, כולל שמירה ב-localStorage.
* שיח חברותא: אזור תגובות ודיונים לכל חידוש בנפרד.
* מחשבון גימטריה: כלי תורני שמחשב ערכים של מילים ומאתר עבורך מיידית חידושים ופסוקים בעלי גימטריה זהה מתוך המאגר.
* התמדה וסימניות: שמירת חידושים מועדפים ומעקב אחר ימי לימוד רצופים (Streak).
* מצב לימוד לילה: מעבר מהיר לעיצוב כהה עם אורות זהב מרגיעים לעיניים ללימוד בשעות המאוחרות.
